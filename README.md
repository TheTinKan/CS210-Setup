This helps the Corner Grocer analyze purchasing trends to make better decisions about store layout.

I organized the program with typical object-oriented principles by creating a seperate class to track the goceries.
I also made sure to implement input validation and formatted the histogram to always display nicely

In the future I could make sure to implement more robust error handling, as well as implementing case insensitive searches.

The biggest challenge was working with the information to output a nice looking histogram.
It took me quite a while to research and then implement how I wanted to implement the format.

This project helped me clear up some issues and misunderstandings I had regarding my skills in file input and output, as well as
help formating some output scenarios.

I made sure to try and keep my code as organized as I could, keeping functionalities fully seperated and splitting my methods
within the corresponding classes. I was also sure to use descriptive names, comments and consistent formatting.
